Welcome, to project magpie!


A compleate and useful simple set of cross platform ANSI C / gcc programs for the cromebook linux shell enviroment.


magpiehelp.c / help.html - Our html help file.


magpie16.c - dumps hex values and contents of file.


statos.c - shows cromebook system stats and other information.
NOTE:statos is a rewrite of another one of my programs made for ubuntu.


magpie.c - search for keywords inside of a file.


magpiescr.c - A text based screen saver that randomly displays jokes.


magpied.c - A text based scrolling file editor.


magpieman.c - shows total system load and other information.


magpies.c - This program will search if our given file name is some type of random encryption type.


tarot.c - A tarot/zork style text card game. Hidden game endings and other random time trap events.


tarotGold.c - A tarot card deck sim. offers command line arguments such as --default3 and/or --default5 for auto fast card readings.


mus.c - shows system size and storage.
NOTE:mus is a linux/ubuntu rewrite for the cromebook as well.


magpiefs.c - scrolling file content dumper and intgrating disassembly functionalities.


magpielife.c - A text based screen saver based on the game of life. A twist to the game Red and Blue life forms are locked into A death match. The Winner eats all.


magpiecpu.c - display the temps of all cpu cores in A system. This fully works on [orange PI Zero 3] check source code notes for testing on cromebook.


magpieavg.c - displays all the core cpu temps in real time. This works on [orange PI Zero 3] check source code notes for testing on cromebook.


magpiesim.c - A player vs cpu game of numbers text based on the game of life.


magpiedir.c - A basic directory listing and search tool.


magpiell.c - This directory listing tool displays files, sizes, groups, users and attributes of files./p>
