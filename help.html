Welcome, to project magpie!


A compleate and useful simple set of cross platform ANSI C / gcc programs for the cromebook linux shell enviroment.


magpiehelp.c / help.html - Our html help file.


magpie16.c - dumps hex values and contents of file.


statos.c - shows cromebook system stats and other information.
NOTE:statos is a rewrite of another one of my programs made for ubuntu.


magpie.c - search for keywords inside of a file.


magpiescr.c - A text based screen saver that randomly displays jokes.


magpied.c - A text based scrolling file editor.


magpieman.c - shows total system load and other information.


magpies.c - This program will search if our given file name is some type of random encryption type.


mus.c - shows system size and storage.
NOTE:mus is a linux/ubuntu rewrite for the cromebook as well.


magpiefs.c - scrolling file content dumper and intgrating disassembly functionalities.


tarotgold.c - full tarot card sim. Based on the Gold classic tarot card deck.
